# Superscript

Some Markdown processors allows to use of **Subscript** and **Superscript**.

To create a **superscript**, use one caret symbol (^) before and after the
character.
```
X^2^
```

### Rendered output

X^2^

### HTML syntax

Alternatively, if the Markdown application supports HTML:
```
X<sup>2</sup>
```

### Rendered output

X<sup>2</sup>
