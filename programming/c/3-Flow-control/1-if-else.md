# If-statements

## If

```c
int i = 10;

if (i == 10) {
    printf("i is 10");
} else {
    printf("i isn't 10");
}
```

```c
int i = 99;

if (i == 10)
    printf("i is 10");
else if (i == 20)
    printf("i is 20");
else if (i == 30)
    printf("i is 30");
else
    printf("i is something else");
```
