# Bindings

- `%` - Matching character: `()`, `{}`, `[]`
- `Shift + o` - Alternate visual
- `~` - Toggle capitalization
- `&` - Insert previously typed character

### Buffer

- `Ctrl + ^` - Alternate buffer
- `Ctrl + '` - Last buffer

### Find

- `*` - Find word under cursor
- `#` - Find word under cursor (reverse)

### Integer

- `Ctrl + A` - Increment next
- `Ctrl + X` - Decrement next

### Macro

- `@` - Apply macro
