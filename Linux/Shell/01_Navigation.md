# Navigation commands

| Command        | Description             |
| -------------- | ----------------------- |
| cd <directory> | Change Directory        |
| cd             | Home Directory          |
| cd .           | Current Directory       |
| cd ..          | Parent Directory        |
| cd ~           | Home Directory          |
| cd -           | Previous Directory      |
| history        | Command History         |
| pwd            | Print Working Directory |
