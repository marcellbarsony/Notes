## Ordered list

To create an ordered list, add a line items with numbers followed by periods.

```md
1. First item
2. Second item
3. Third item
4. Fourth item
```

### HTML syntax

```html
<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
  <li>Fourth item</li>
</ol>
```

### Rendered output

1. First item
2. Second item
3. Third item
4. Fourth item

## Unordered list

To create an unordered list, add dashes (-), asterisks (*), or plus signs (+) in front of line items

```md
- First item
- Second item
- Third item
```

### HTML syntax

```html
<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>
```

### Rendered output

- First item
- Second item
- Third item
