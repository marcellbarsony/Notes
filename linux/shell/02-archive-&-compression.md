## Tar (Archive)

| Command                            | Description                       |
| ---------------------------------- | --------------------------------- |
| cf                                 | Tar file                          |
| xf                                 | UnTar file                        |
| c                                  | Create archive                    |
| f                                  | Specify filename                  |
| j                                  | bzip2 compression                 |
| k                                  | Don't overwrite                   |
| t                                  | Table of contents                 |
| v                                  | Verbose                           |
| w                                  | Ask for confirmation              |
| x                                  | Extract                           |
| z                                  | Zip/Gzip                          |

## Gunzip (Compression)

| Command                            | Description                       |
| ---------------------------------- | --------------------------------- |
| gzip                               | Compress file                     |
| gzip -d                            | Decompress file                   |
